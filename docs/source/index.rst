Alfred3 DBTools Documentation
=============================

**Alfred3_dbtools** is an extension for `alfred3 <https://github.com/ctreffe/alfred>`_, a Python library for developing dynamic and interactive experiments in the social sciences. To install it, simply run

.. code-block:: console

    $ pip install alfred3_dbtools
